The Project Needs PHP 7.2 and Composer in order to run properly.

The file script.php should be executed with php and it accepts one argument - the path to input file (ex. input.csv).

The Commands should be executed from the Terminal in the Project Root Directory.

Run Program Command:
php script.php input.csv

Run Tests Command:
./vendor/bin/phpunit --bootstrap vendor/autoload.php tests/
